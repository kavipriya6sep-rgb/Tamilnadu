Culture of Tamil Nadu – HTML Project
📌 Project Description

This project is a simple HTML webpage that describes the rich culture and heritage of Tamil Nadu. The webpage includes information about traditional dance, festivals, food, dress, and famous temples of Tamil Nadu.

It is created using only basic HTML tags without CSS or JavaScript.

🛠 Technologies Used

HTML5

Basic HTML tags (heading, paragraph, image, list, etc.)

📖 Sections Included

About Tamil Nadu

Bharatanatyam (Traditional Dance)

Pongal (Famous Festival)

Traditional Food

Traditional Dress

🎯 Project Objective

The main objective of this project is to:

Practice basic HTML structure

Understand how to use headings, paragraphs, images, and lists

Create a simple cultural information webpage

🖼 Features

Clean and simple layout

Informative cultural content

Images included using external links

Fully beginner-friendly HTML project

🚀 How to Run the Project

Download or copy the HTML file.

Save it as index.html.

Open the file in any web browser (Chrome, Edge, etc.).

✨ Output

The webpage displays information about Tamil Nadu culture along with relevant images in a structured format.
